# LINE公式アカウント 初回ステップ配信 プロンプトテンプレート

## 使い方

1. 下記「サロン情報入力欄」に自サロンの情報を記入する
2. プロンプト全文をコピーしてChatGPT / ClaudeなどのAIツールに貼り付ける
3. 出力されたメッセージを確認・微調整してLINEに登録する

---

## プロンプト本文

```
あなたは美容サロンのLINE公式アカウント運用の専門家です。
以下のサロン情報をもとに、新規フォロワーへの初回ステップ配信メッセージを5通作成してください。

【サロン情報】
- サロン名：
- 所在地（都道府県・市区町村）：
- 得意メニュー・専門技術：
- ターゲット顧客（年齢層・悩み・ライフスタイルなど）：
- サロンの雰囲気・こだわり：
- 客単価の目安：
- 予約方法（電話 / WEB / LINE など）：
- キャンペーン・特典（あれば）：

【各メッセージの条件】
- 配信タイミング：登録直後 / 翌日 / 3日後 / 7日後 / 14日後
- 文字数：各メッセージ150〜250文字程度
- トーン：親しみやすく温かみのある文体（硬すぎず、馴れ馴れしすぎない）
- 絵文字：1メッセージあたり2〜4個を自然に使用
- 各メッセージの末尾には予約や問い合わせへの導線（CTA）を入れる

【各メッセージのテーマ】
1通目（登録直後）：フォローのお礼と自己紹介・サロンの特徴をひと言で
2通目（翌日）：得意メニューや技術のこだわりを伝える
3通目（3日後）：お客様の悩みに共感し、サロンで解決できることを伝える
4通目（7日後）：お客様の声・ビフォーアフターのイメージ・信頼感を醸成する
5通目（14日後）：初回来店への背中を押すオファーまたは予約のご案内

出力形式：
【1通目 - 登録直後】
（メッセージ本文）

【2通目 - 翌日】
（メッセージ本文）

…以下同様
```

## 記入例

| 項目 | 入力例 |
|------|--------|
| サロン名 | ヘアサロン Bloom |
| 所在地 | 大阪府堺市 |
| 得意メニュー | 白髪染め・頭皮ケア・カラーリタッチ |
| ターゲット顧客 | 40〜60代女性・白髪や頭皮の悩みを持つ方 |
| サロンの雰囲気 | 完全予約制・1対1でゆったりした空間 |
| 客単価 | 8,000〜12,000円 |
| 予約方法 | LINEまたはお電話 |
| キャンペーン | 初回限定トリートメント無料サービス |

---

## 調整のポイント

- 出力後、サロン名・固有メニュー名・地名など具体的な情報が正しく入っているか確認する
- CTAのリンク先（予約URL・電話番号）は手動で追記する
- トーンが合わない場合は「もう少し柔らかく」「もう少しプロフェッショナルな文体で」と追加指示する
- LINE公式アカウントの文字制限（1メッセージ500文字）に注意する
