# LINE公式アカウント あいさつメッセージ テンプレート

## 概要

あいさつメッセージは、ユーザーがLINE公式アカウントを友だち追加した直後に自動送信されるメッセージです。
第一印象を左右する重要なメッセージのため、サロンの魅力と次のアクションを簡潔に伝えることが大切です。

---

## テンプレートA（スタンダード型）

> こんにちは！【サロン名】です✨
> 友だち追加ありがとうございます😊
>
> このアカウントでは、
> ・お得なキャンペーン情報
> ・ヘアケア・スタイルのお役立ち情報
> ・ご予約の受付
>
> などをお届けします。
>
> ご予約・ご質問はこのLINEからいつでもどうぞ！
> スタッフ一同、お会いできる日を楽しみにしています💐
>
> ▼ご予約はこちら
> 【予約URL または「予約」と送信してください】

## カスタマイズ項目

| 置き換え箇所 | 内容 |
|-------------|------|
| 【サロン名】 | 自サロンの名称 |
| 【担当者名】 | オーナー名またはスタッフ名 |
| 【予約URL】 | ホットペッパー・自社サイト等のURL |
| 得意メニュー | サロンの専門・強みに合わせて変更 |
| 絵文字 | サロンのトーンに合わせて調整（または削除） |

---

## 設定上の注意

- あいさつメッセージはLINE公式アカウントマネージャーの「応答設定」＞「あいさつメッセージ」から設定
- 文字数上限：500文字（テンプレートはすべて範囲内）
- 画像やリッチメッセージと組み合わせると視覚的な印象が上がる
- 設定後は必ず自身のアカウントで友だち追加して表示を確認する
